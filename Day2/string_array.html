Methods of Strings and Array(Remaining)

(I)Strings

1.Substring-The substring() method extracts the characters from a string, between two specified indices, and returns the new sub string.
For Example: let a= 'Hello everyone';
console.log(a);
console.log(a.substring(1,4));
console.log(a.substring(1));
console.log(a.substring(-3));
console.log(a.substring(5,0));
Note: Press F12 for the output
Output: Hello everyone
ell
ello everyone
Hello everyone
Hello
2.Substr- The substr() method extracts parts of a string, beginning at the character at the specified position, and returns the specified number of characters.
For Example: let b= 'Hello everyone';
console.log(b);
console.log(b.substr(1,8));
console.log(b.substr(3));
console.log(b.substr(-3,7));
console.log(b.substr(4,2));
Note: Press F12 for the output
Output: Hello everybody
ello ev
lo everybody
Hello e
o
3.Slicing-The slice() method extracts parts of a string and returns the extracted parts in a new string.Use the start and end parameters to specify the part of the string that you want to extract.
For Example: let c= 'Good morning';
console.log(c);
console.log(c.slice(3,6));
console.log(c.slice(-3));
console.log(c.slice(-7,-2));
console.log(c.slice(4));
Note: Press F12 for the output
Output: Good morning
d m
ing
morni
morning
4.replace-The replace() method searches a string for a specified value,or a regular expression, and returns a new string where the specified values are replaced.
For Example: let d= 'Mr Blue has Blue car and Blue house.';
console.log(d);
console.log(d.replace(/Blue/g,'Green'));
console.log(d.replace('car','bike'));
Note: Press F12 for the output
Output:Mr Blue has Blue car and Blue house. Mr Green has Green car and Green house. Mr Blue has Blue bike and Blue house.

(II)Arrays

1.splice method-The splice() method adds/removes items to/from an array, and returns the removed item(s).
For examplelet fruits = ["Banana", "Orange", "Apple", "Mango"];
console.log(fruits);
console.log(fruits.splice(2, 1, "Lemon", "Kiwi"));
console.log(fruits.splice(2, 2));
Note: Press F12 for the output
2.to String method-The toString() method returns a string with all the array values, separated by commas.
For examplelet num = [1,2,3,4,5];
console.log(num);
console.log(num.toString());
Note: Press F12 for the output
3.Includes method-The includes() method determines whether an array contains a specified element.This method returns true if the array contains the element, and false if not.
For examplelet n = [1,2,3,4];
console.log(n);
console.log(n.includes(3));
Note: Press F12 for the output
