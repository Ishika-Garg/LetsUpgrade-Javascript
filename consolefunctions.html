Console Functions

1. console.log() Function

This function is mainly used to log(print) the output to the console. We can put any type inside the log(), be it a string, array, object, boolean etc.
For Example: console.log([1,2,3,4])
Note: Press F12 for showing the output.
Output: In the browser console,the output is shown as:
array(4)
0:1
1:2
2:3
3:4

2. console.error() Function

It is used to log error message to the console. Useful in testing of code. By default the error message will be highlighted with red color.
For Example: console.error('This is a simple error')
Note: Press F12 for showing the output.
Output: In the browser console,the output is shown as:
This is a simple error(in red colour)

3. console.warn() Function

It is used to log warning message to the console. Useful in testing of code. By default the error message will be highlighted with yellow color.
For Example: console.warn('This is a warning message')
Note: Press F12 for showing the output.
Output: In the browser console,the output is shown as:
This is a warning message(in yellow colour)

4. console.clear() Function

It is used to clear the console. The console will be cleared, in case of Chrome a simple overlayed text will be printed like : ‘Console was cleared’ while in firefox no message is returned.
For Example: console.clear()
Note: Press F12 for showing the output.
Output: In the browser console,the output is shown as:
Console was cleared

5. console.table() Function

This method allows us to generate a table inside a console. The input must be an array or an object which will be shown as a table.
For Example: console.table({'a':1, 'b':2, 'c':3})
Note: Press F12 for showing the output.
Output: In the browser console,the output is shown as a table

6. Custom console log Function

It is used to add Styling to the console logs in order to make logs Custom . The Syntax for it is to add the css styling as a parameter to the logs which will replace %c in logs.
For Example: console.log('%c A message involving javascript','color:green')
Note: Press F12 for showing the output.
Output: In the browser console,the output is shown as:
A message involving javascript(in green colour)

7. console.time() and console.timeEnd() Function

Whenever we want to know the amount of time spend by a block or a function, we can make use of the time() and timeEnd() methods provided by the javascript console object. They take a label which must be same, and the code inside can be anything( function, object etc)
For Example: console.time('Time taken');
console.log([1,2,3,4]);
console.error('This is a simple error');
console.warn('This is a warning message');
console.table({'a':1, 'b':2, 'c':3});
console.timeEnd('Time taken'); Note: Press F12 for showing the output.
Output: In the browser console,the output is shown as:
Time taken: 0.673095703125ms

8. console.group() and console.groupEnd() Function

group() and groupEnd() methods of the console object allows us to group contents in a separate block,which will be indented.Just like the time() and the timeEnd() they also accepts label, again of same value.
For Example: console.group('Group created'); console.warn('warning!'); console.error('error here'); console.log('4466'); console.groupEnd('Group created'); console.log('new section'); Note: Press F12 for showing the output.
Output: In the browser console,the output is shown as a group named Group created
